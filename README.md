## Full Stack Blog

Full-stack blog platform developed with Next.js, TypeScript, Tailwind CSS, Prisma, Better Auth, and full CRUD functionality.

[View Live Demo](https://blogapp.hugo-miranda.dev)

## About The Project

![blog Screenshot](https://res.cloudinary.com/dck0d5qwp/image/upload/v1766248110/Novo_Projeto_4_zfuafk.png)

Blog Next is a full-stack blogging platform built with Next.js, TypeScript, Tailwind CSS, and Prisma.

## Key Features & Functionality

- **Full CRUD (Create, Read, Update, Delete):** Users can create new blog posts, edit existing ones, view all posts, and delete content safely.
- **User Authentication:** Integration with **BetterAuth**, supporting login via email, Google, and GitHub.
- **Dynamic Routes & SEO Optimization**
- **Responsive Design**
- **Database Integration:** Uses **Prisma** to manage the database efficiently, demonstrating knowledge of data modelling, migrations, and query handling.
- **TypeScript** for type safety
- **Modular and reusable React components**

### Installation

1. Clone the repo

```sh
git clone https://github.com/hugolomba/blog-next.git
```

2. Install packages

```sh
npm install
# or
yarn install
# or
pnpm install
```

3. Set .env variables

```sh
PRISMA_DATABASE_URL=""
POSTGRES_URL=""
PRISMA_DATABASE_URL=""

DATABASE_URL=
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
BETTER_AUTH_GITHUB_ID=
BETTER_AUTH_GITHUB_SECRET=

BETTER_AUTH_GOOGLE_ID=
BETTER_AUTH_GOOGLE_SECRET=

NEXT_PUBLIC_RICH_TEXT_EDITOR_API_KEY=

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
NEXT_PUBLIC_CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
